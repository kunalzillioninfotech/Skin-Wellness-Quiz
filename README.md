🏗️ Tech Stack\n
Frontend\n
React (Vite)\n
Tailwind CSS\n
Axios\n
Lucide Icons\n
React Hot Toast\n

Backend\n
Node.js\n
Express.js\n
MySQL\n
Multer (File Upload)\n
Cloudinary (Image Hosting)\n


⚙️ Installation & Setup\n
1️⃣ Clone Repository\n
git clone https://github.com/your-username/skin-quiz.git\n
cd skin-quiz\n
🖥️ Frontend Setup\n
cd frontend\n
npm install\n
npm run dev\n

👉 Runs on: http://localhost:5173\n

🔧 Backend Setup\n
cd backend\n
npm install\n
npm run dev\n

👉 Runs on: http://localhost:5000\n

🔐 Environment Variables\n

Create a .env file inside backend/\n

PORT=5000\n

# MySQL Config\n
DB_HOST=localhost\n
DB_USER=root\n
DB_PASSWORD=yourpassword\n
DB_NAME=skin_quiz

# JWT (if used)\n
JWT_SECRET=your_secret_key\n

# Cloudinary Config\n
CLOUD_NAME=your_cloud_name\n
CLOUD_API_KEY=your_api_key
CLOUD_API_SECRET=your_api_secret
